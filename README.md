# Portafolio de Proyectos Python

¡Hola! Soy **Daniel Tascon**. Este repositorio contiene proyectos prácticos que he desarrollado mientras aprendo Python, enfocados en resolver problemas reales con código.

## Mi Enfoque
Crear soluciones funcionales que demuestren mi comprensión de los fundamentos de la programación, desde la lógica hasta el manejo de datos.

## Proyectos

| Proyecto | Descripción | Tecnologías Clave | Lo que Demuestra |
| :--- | :--- | :--- | :--- |
| **[Sistema de Inventario](./Sistema_de_inventario)** | CRUD completo para gestionar productos en memoria. | Listas, Diccionarios, Lógica de Menú | Estructura de datos y lógica de negocio. |
| **[Agenda de Contactos](./agenda-contactos)** | Gestor de contactos con persistencia en archivo `.txt`. | Manejo de Archivos (`open/read/write`), Persistencia | Entrada/Salida de datos y almacenamiento permanente. |

## Cómo Explorar
1.  Haz clic en el nombre de cualquier proyecto de la tabla de arriba.
2.  Lee su `README.md` específico para entender su propósito y lo que aprendí.
3.  Revisa el código fuente en los archivos `.py`.
