# Catálogo de Productos en Python

Este proyecto implementa un sistema básico de gestión de inventario con operaciones CRUD:

- Agregar productos
- Mostrar catálogo completo
- Buscar por nombre
- Actualizar datos de un producto
- Eliminar productos

## Requisitos
- Python 3.10 o superior

## Ejecución
```bash
python catalogo.py
